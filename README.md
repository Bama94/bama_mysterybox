# 🎁 Mystery Box Script (QBox / QBX Core)

A Call of Duty: Zombies–inspired Mystery Box system built for **QBox / QBX Core**, designed to bring high-energy gameplay, randomness, and replayability into your server.

This isn’t just a loot box — it’s a full gameplay experience with effects, sounds, and dynamic systems that make every spin feel alive.

---

## 🔧 Core Features

### ✨ Multiple Box Locations  
Configure unlimited spawn points across your map. The system handles rotation and spawning automatically.

### 📦 Fully Configurable Props
- Unopened box prop  
- Opened box prop  
- Placeholder props for inactive locations (shows potential spawn spots)

### 🎯 Advanced Loot Pool System  
Add/remove weapons with a weighted chance system to fully control rarity and balance.

### 🧸 Teddy Bear System (Classic Zombies Feature)  
Add multiple teddy bear entries that can randomly trigger a box relocation.

More teddy bears = more chaos.

### 💰 Economy Integration  
Set a configurable price per spin to match your server economy.

### 🗺️ Dynamic Location Feedback  
Inactive spawn points show placeholder props so players can visually track possible box locations.

### 🔊 Immersive Audio & Effects  
Custom sounds and effects for:
- Opening the box  
- Receiving rewards  
- Teddy bear trigger events  
- Box relocation  

Fully customizable to match your server vibe.

### 🎬 Animation Ready System  
Built with future expansion in mind for custom animations:
- Box interactions  
- Teddy bear events  
- Weapon pulls
